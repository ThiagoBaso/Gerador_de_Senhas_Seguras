# Secure Password Generator

Aplicação para geração de senhas seguras com critérios configuráveis.

## 🚀 Funcionalidades
- Geração de senha com tamanho personalizado
- Suporte a letras, números e símbolos
- Uso de aleatoriedade criptograficamente segura
- Embaralhamento com algoritmo Fisher-Yates

## 🛠️ Tecnologias
- React
- TypeScript
- Web Crypto API

## 📦 Como rodar
```bash
npm install
npm run dev
